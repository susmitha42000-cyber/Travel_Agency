# Stackly Futuristic Travel Agency

Stackly is a complete static travel agency website built with HTML5, CSS3, vanilla JavaScript, local AOS-style scroll animations, CSS 3D transforms, and a custom canvas particle background.

## Pages

- `index.html` - home page with hero, 3D globe, destinations, packages, stats, testimonials, gallery preview, CTA, newsletter, and footer
- `destinations.html` - filterable destination cards
- `packages.html` - sortable premium tour packages
- `about.html` - company story and mission
- `services.html` - travel service overview
- `gallery.html` - masonry gallery with lightbox
- `testimonials.html` - customer testimonials
- `contact.html` - contact form with validation feedback
- `login.html` - client login form
- `signup.html` - registration form with searchable country selector
- `forgot-password.html` - password reset form

## Features

- Premium dark theme with glassmorphism, neon highlights, and responsive layouts
- Sticky responsive navigation with mobile hamburger menu
- Local AOS-compatible scroll reveal implementation in `vendor/aos`
- Live particle background in `vendor/particles`
- CSS 3D tilt cards, animated travel globe, floating chips, moving clouds, and airplane animation
- JavaScript counters, gallery lightbox, package sorting, destination filtering, newsletter feedback, and form validation
- Semantic page structure, SEO metadata, accessible labels, responsive constraints, and reduced-motion support

## Usage

Open `index.html` directly in a browser. No build step, package manager, backend, React, Bootstrap, or Tailwind is required.

## Assets

The site uses optimized remote Unsplash images for rich visual presentation and includes the requested local asset folders for future project-owned images, icons, flags, and videos.
